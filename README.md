📖 Description

This project is a simple e-commerce website focused on watches. It includes product listings, a clean user interface, and basic shopping features. The goal of this project is to learn how to design responsive websites and implement interactive elements using JavaScript.

✨ Features

Responsive design (Mobile + Desktop)
Product listing with images, price, and ratings
Add to Cart functionality
Simple product filtering (category/price)
Clean UI with hover effects and animations
Easy navigation

🧰 Tech Stack
HTML
CSS
JavaScript

📚 What I Learned

Structuring a complete website
Creating reusable UI components
Making responsive designs
Adding interactivity with JavaScript
Understanding basic e-commerce workflow

🔧 Future Improvements

Add user authentication (login/signup)
Improve cart functionality
Add search feature
Connect backend (database integration)
Payment system integration
